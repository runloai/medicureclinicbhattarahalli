# Medicure Clinic & Medical

Static website for Medicure Clinic & Medical, Bhattarahalli.

## Deploy on Netlify

1. Connect the GitHub repository to Netlify.
2. Use the repository root as the publish directory.
3. Leave the build command empty.
4. Enable Netlify Forms to collect appointment requests from `index.html`.

## Files

- `index.html` - main one-page clinic website
- `styles.css` - responsive visual system based on the Stitch design
- `script.js` - mobile navigation, status text, and date input behavior
- `assets/` - local image and favicon assets
- `netlify.toml` - Netlify publish and security header config

The original Stitch export files are intentionally ignored by Git.
